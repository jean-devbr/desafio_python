# Desafio Python

Este repositório contém uma coleção de desafios e exercícios em Python, organizados por níveis de dificuldade.

## Estrutura

- **jogoaleatorio/**: Desafios específicos.
- **Nivel 1/**: Exercícios básicos.
- **Nivel 2/**: Exercícios intermediários.
- **Nivel 3/**: Exercícios avançados, incluindo subpastas com variações de problemas.

## Como executar

Para executar um exercício, navegue até o diretório correspondente e use o comando:

```bash
python <nome_do_arquivo>.py
```

Certifique-se de ter Python instalado em seu sistema.
